# Minikube Notes

## Install and start Minikube

1. Download the minikube package
```bash
$ curl -LO https://storage.googleapis.com/minikube/releases/latest/minikube-latest.x86_64.rpm
$ sudo rpm -Uvh minikube-latest.x86_64.rpm
```
2. Start minikube
```bash
minikube start
```
